# MRDVS V2 Pro
### Infrastructure-Free Fusion-SLAM™ Localization Module for AGVs, AMRs & Industrial Vehicles

The **V2 Pro** is an all-in-one, embedded real-time localization module engineered for the modern digital warehouse. Combining a 2D LiDAR, ceiling-forward vision sensor and IMU in a single factory-calibrated unit, it delivers centimeter-accurate positioning without requiring UWB anchors, reflectors, AprilTags, or magnetic strips.

Powered by our decade-refined **Fusion-SLAM™** technology—a tightly-coupled LiDAR-Visual-Inertial SLAM algorithm with deep-learning enhancement—the V2 Pro provides reliable **±3 cm accuracy at 20 Hz** in challenging environments including high-bay racking, narrow aisles, metallic structures, and dynamic storage layouts.

---

## Technical Specifications

| Parameter | Specification |
|-----------|--------------|
| **Positioning Accuracy** | ±3 cm (up to 15m ceiling height) |
| **Update Rate** | 20 Hz |
| **Latency** | &lt;100 ms (sensor to pose output) |
| **Sensors** | 2D LiDAR (270° FoV, 40m range), IR Camera (108°×57° FoV, 8 IR emitters), 6-axis IMU |
| **Processing** | Onboard edge computing, no external server required |
| **Map Capacity** | Single map &gt;100,000 m²; Total storage &gt;1,000,000 m² |
| **Power** | 24V DC / 3A |
| **Interfaces** | Gigabit Ethernet, CAN Bus (CANopen), Wi-Fi 2.4GHz, 4x Digital I/O, Audio |
| **Environmental** | IP54, -20°C to +60°C operating, shock-resistant for forklift mounting |
| **Dimensions** | 146.5 × 66.3 × 89.4 mm (excl. cables) |
